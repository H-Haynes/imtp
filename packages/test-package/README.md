# @imtp/test-package

这是 test-package 包的说明文档。

## 安装

```bash
npm install @imtp/test-package
```

## 使用方法

```typescript
import { hello } from '@imtp/test-package';

console.log(hello('World')); // 输出: Hello from test-package, World!
```

## API

### `hello(name: string): string`

返回一个问候字符串。

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 测试
pnpm test
```
